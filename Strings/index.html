Double Data Type:

Double’s work exactly as int’s except that they can be fractional. This means we can have a variable that holds a decimal number, like 2.5. This is not possible with int’s.




First thing to note here is that just as the keyword ‘int’ was necessary to make an int variable, the keyword ‘double’ is necessary to make a double variable.

Here we initialized a variable called ‘pi’ to a decimal value and printed it out. As you can see, it works exactly as expected. Then when we initialized a new variable, ‘whole_number’, to 3 and the output was show as 3.0. This is because ‘whole_number’ is of type double, which means even if you assign an integer to it, it will still be printed in decimal form.

One more important thing to know before we end this section is that it is illegal to assign a double to an int. Meaning, once you declare an int, you can only put an int inside of it, not a double. If you try to assign a decimal number to an int, you will get an error and the program will not compile.




If we try to run the above code, we get the following error: Error:(4, 19) java: incompatible types: possible lossy conversion from double to int

All this is saying is that the data types are incompatible, and you cannot convert a double (5.812 in this case) to an int.
