Mathematical Operations :

There are four basic mathematical operations you can do in Java: addition, subtraction, multiplication and division. There are specific symbols you must use for each of them.

Addition: +
Subtraction: -
Multiplication: *
Division: /

We can use the above four symbols to do basic math operations on variables in java, as demonstrated below:





We used each of the 4 operators on the variables ‘a’ and ‘b’, assigned the result of that operation into another variable, and then printed each of those results out. Of course here we used int, but these operators would work exactly the same with double data type variables.

Concatenation:

The plus operator can also be used with String data type. The expected question here would be: but how can you add two strings? Well you can’t. The plus operator, when used with strings, does something completely different than what it does with ints and doubles. With strings , the plus operator performs concatenation.
