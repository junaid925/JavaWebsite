Introduction to Java



The above Java code is a simple “Hello World” program designed to introduce you to the bare basics of the language.

Line 1 opens what’s called a class in Java. In Java, everything is encased within a class. Since this website is for beginners, we will not go into the details of what a class is or how it works. For now, simply accept that all code you write will be inside a class.

Line 3 opens up our main method. A method is a block of code (opened and closed with curly braces as shown above) that performs a specific task. All Java programs begin with the default main method, and can then jump into other methods. We will learn how to make other methods of our own later in these tutorials, but for now we will only focus on main.

All methods have a header which indicates its name and parameter(s), and the header for main will always look as it does in line 3. You do not need to understand what the syntax of line 3 does for now, we will get into that later in these tutorials. All code you write for this method will be after the opening brace and before the closing brace.

In this case, we have only one line of code in our main method. All it does is print the words “Hello World!” onto the console. Once we run the program we see the console looks like this:



As you can see, line 4 simply prints the text into double quotes onto the console. In fact, anytime you need to print any text onto the console, you must write System.out.println(), and include whatever text you want to print inside the parenthesis surrounded by double quotes. This print commands send your text into the console.

Notice this time we printed two lines of text, and thus used two separate print commands. As you can see, you can print both words and number, just as long as you surround them with double quotes. This is why the second print command (this list of numbers) prints on a new line. If we were to have the third print command, it would print the text on a new line, directly under the numbers.
