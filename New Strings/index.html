String Data Type:

String data types allows you to hold non-numerical values, such as letters and symbols. Data like names and phrases are stored in String variables.

To declare a String variable, simply write the keyword String followed by the variable name. You can then initialize it to a non-numerical value such as a name or even an entire sentence. Char Data Type:

Char data type is very similar to String, except it can hold only a single character. You make a char by writing the keyword char, followed by the variable name, and then assigning it a value, but this time in single-quotes.
